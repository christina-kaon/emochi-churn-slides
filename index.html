09行动建议

三个优先级

P0

错误率治理

  • receive_timeout 和 receive_error 是主要错误类型

  • 重点治理长 context RP 会话中的错误——剧情中断影响极大

  • 目标：将 >5% 错误率用户占比从 28% 降到 15%

P1

Bot 推荐优化

  • D31 用户试 107 个 bot——推荐系统没有帮他们找到匹配

  • 首周主动推荐高匹配度 RP bot，减少用户探索成本

  • 建立"Bot-User 匹配度"指标——深度对话 = 好匹配

P2

续费挽留

  • Cancel flow 提示"订阅 X 日到期，此前仍可使用"

  • 到期前 3 天推送最佳体验回忆——"你创造了 XX 个故事"

  • 提供"到期取消"替代"立即取消"选项
